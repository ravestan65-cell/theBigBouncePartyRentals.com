# The Big Bounce Party Rentals

Official website for **The Big Bounce Party Rentals** — premium bounce houses, water slides, and party rentals shipped same day across the USA.

Website: [thebigbouncepartyrentals.com](https://thebigbouncepartyrentals.com)

## Project Structure

```text
/

├── src/
│   ├── components/
│   ├── data/
│   ├── layouts/
│   ├── lib/
│   ├── pages/
│   └── styles/
└── package.json
```

## Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |

Built with [Astro](https://astro.build).
